Satellite Applications Catapult - Public Data

This repository holds the Public Data Set of the Satellite Applications Catapult. However, access is restricted by means of a white-listing policy.
To access these files with an AWS S3 client the name of the AWS bucket to use is public-eo-data, and the endpoint to use is s3-uk-1.sa-catapult.co.uk.
